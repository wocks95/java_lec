if 문
  1. 형식
    if(조건식) {
      본문
    }
  2. if 문의 조건식의 결과가 true 이면 본문을 실행한다.
  3. 본문은 중괄호{}로 묶는다.
  4. 본문이 1개 코드로 구성되는 경우 중괄호{}를 생략할 수 있다.

else if 문
  1. 형식
    if(조건식) {
      본문
    } else if(조건식) {
      본문
    } else if(조건식) {
      본문
    }
  2. if 문 이후에 추가할 수 있는 선택 구문이다.
  3. if 문의 조건식의 결과가 false 이면 else if 문의 조건식이 실행된다.
  4. 본문은 중괄호{}로 묶는다.
  5. 본문이 1개 코드로 구성되는 경우 중괄호{}를 생략할 수 있다.
  6. 원하는 만큼 추가해서 사용할 수 있다.

else 문
  1. 형식
    if(조건식) {
      본문
    } else if(조건식) {
      본문
    } else {
      본문
    }
  2. if 문 또는 else if 문 이후에 딱 한 번 나타날 수 있는 선택 구문이다.
  3. if 문과 else if 문의 모든 조건식의 결과가 false 이면 else 문의 본문을 실행한다.
  4. 별다른 조건식을 작성하지 않음에 주의한다.

switch 문 / break 문
  1. 표현식의 결과값에 따라 분기 처리한다. (조건식이 아님)
  2. 표현식의 결과값은 byte, short, int, char, String 중 하나여야 한다. (boolean, float, double, long 타입은 불가능하다.)
  3. 표현식의 결과값이 일치하는 첫 번째 case 부터 모두 다 실행한다.
  4. switch 문의 실행 종료를 위해서 break 문을 사용한다.
  5. 모든 case 문이 작성된 이후 마지막에 default 문을 추가할 수 있다.
  6. 일반적으로 if 문에 비해서 가독성이 좋다.
  7. 형식
     switch(표현식) {
     case 결과값1:
       실행문
     case 결과값2:
       실행문
     ...
     default:
       실행문
     }
