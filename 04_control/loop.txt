while 문
  1. 형식
    while(조건식) {
      본문
    }
  2. while 문의 조건식의 결과가 true 이면 본문을 실행한 다음 다시 while 문의 조건식으로 돌아간다.
  3. while 문의 조건식의 결과가 false 이면 본문을 실행하지 않고 while 문을 종료한다.
  4. 본문이 1개 코드로 구성되는 경우 중괄호{}를 생략할 수 있다.
  5. while 문의 본문이 한 번도 수행되지 않을 수도 있다.

do while 문
  1. 형식
    do {
      본문
    } while(조건식);  // 세미콜론(;) 필요
  2. 조건식의 만족 여부와 상관 없이 최초 1회는 항상 본문을 수행한다.
  3. 나머지 다른 특징은 while 문과 동일하다.

for 문
  1. 형식
    for(초기상태; 조건식; 상태변화) {
      본문
    }
  2. 초기상태를 처리한 뒤 조건식으로 이동한다.
  3. 조건식의 결과가 true 이면 본문을 실행한 뒤 상태변화를 처리한다.
  4. 상태변화를 처리한 뒤 다시 조건식으로 이동한다.
  5. 조건식의 결과가 false 이면 본문을 실행하지 않고 for 문을 종료한다.
  6. 본문이 1개 코드로 구성되는 경우 중괄호 {}를 생략할 수 있다.
 
break 문
  1. switch 문을 종료할 때 사용한다.
  2. while 문이나 for 문 같은 반복문을 종료할 때 사용한다.

continue 문
  1. while 문이나 do while 문의 본문에서 continue 문을 만나면 곧바로 조건식으로 이동한다.
  2. for 문의 본문에서 continue 문을 만나면 곧바로 상태변화로 이동한다.
  3. continue 문 이후의 본문 코드를 수행하지 않기 위해서 사용한다.
